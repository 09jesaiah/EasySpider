## Note

Remember to update EasySpider.crx everytime the extension updates.



## Run Instruction

```
npm install
npm install @electron-forge/cli
```

## Package Instruction

```
npx electron-forge import
npm run package
```

optional:

```
npm run make
```

## For windows x64
打开下面两个cmd即可打包
```
package_win64.cmd
clean_win64.cmd
```
